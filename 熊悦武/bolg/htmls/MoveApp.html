移动开发与适配
一： 第一种方案是：所有的单位使用rem来适配；首先在页面上设置html的font-size的大小，比如：html { font-size: 100px; }
  • 
    @media(min-width: 320px) { html { font-size: 1em; } }
  • 
    @media(min-width: 360px) { html { font-size: 1.5em; } }
  • 
    @media(min-width: 400px) { html { font-size: 2em; } }
  • 
    @media(min-width: 640px) { html { font-size: 2.5em; } }
