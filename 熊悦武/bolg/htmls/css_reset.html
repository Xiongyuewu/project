解决css代码冗余
一.最简化的CSS Reset(重设) :
			*{
			    outline: 0px;//鼠标点击输入框不在出现蓝色的框框
			    padding: 0px;//设置内边距为0px
			    margin: 0px;//设置外边距为0px
			    border: 0px;//设置边框为0px
			  }
二.浓缩实用型CSS Reset(重设):
			*{
		       vertical-align: baseline;//veritical是垂直的意思，而align是对齐的意思，两个合起来就是在垂直方向进行对齐。
		       font-weight: inherit;//字体加粗
		       font-family: inherit;//字体类型
		       font-style: inherit;//字体风格
		       font-size: 100%;//字体大小
		       outline: 0;//鼠标点击输入框不在出现蓝色的框框
		       padding: 0;//设置内边距为0px
		       margin: 0;//设置外边距为0px
		       border: 0;//设置边框为0px
			  }
三.Poor Man 的CSS Reset:
			html, body { 
			       padding: 0; 
			       margin: 0; 
			}
			html {
			       font-size:1em;//1em=16px
			} 
			body {
			       font-size:100%;
			} 
			a img, :link img, :visited img {
			       border:0px;
			}
